## Download

```bash
# Clone this repository
$ https://github.com/naddurkostia/LCS-LinuxShell.git

# Go into the repository
$ cd LCS-LinuxShell

# Run the app
$ ./a.out
```

> **Note**
> The shell is always being updated, so stay tuned!
